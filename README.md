# NextStep AI — AI-Powered Career Coach

Welcome to **NextStep AI**, your personal AI-driven career companion. This platform helps you build professional resumes, generate cover letters, prepare for interviews, and gain industry insights using state-of-the-art AI.

## 📖 Project Documentation

For a detailed breakdown of the project architecture, tech stack, and features, please refer to:
👉 **[project.md](./project.md)**

## 🚀 Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Setup environment variables:**
   Create a `.env` file based on the keys listed in `project.md`.

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Visit:** [http://localhost:3000](http://localhost:3000)

## 🛠️ Tech Highlights

- **Next.js 15** & **Tailwind CSS**
- **MongoDB** & **Mongoose**
- **NextAuth v5** (Authentication)
- **Google Gemini AI** & **OpenRouter**
- **Framer Motion** (Animations)
